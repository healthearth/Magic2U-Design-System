The Magic2U Design System

Magic2U is a modular, reusable design system and engineering platform. It includes UI components, design tokens, workflows, AI plugins, and a CLI — all designed to help teams build consistent, scalable digital products.

  • ✨ Web Components (<magic-card>, <magic-button>)
  • ✨ Design tokens for color, spacing, typography
  • ✨ Workflow engine
  • ✨ AI plugins (OpenAI, LLM, Weather)
  • ✨ CLI for scaffolding apps
