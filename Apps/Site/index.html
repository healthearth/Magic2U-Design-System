The Magic2U Design System

Magic2U is a modular, reusable design system and engineering platform. It includes UI components, design tokens, workflows, AI plugins, and a CLI.

  • ✨ Web Components (<magic-card>, <magic-button>)
  • ✨ Design tokens for color, spacing, typography
  • ✨ Workflow engine
  • ✨ AI plugins
  • ✨ CLI for scaffolding apps
