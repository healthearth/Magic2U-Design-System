# Magic2U Demo Applications

This folder contains two simple applications that demonstrate the Magic2U design system in action.

## 1. Metrics Dashboard

Shows how Magic2U components work together:

- Data components fetch sample metrics
- AI components summarize the data
- Workflow components orchestrate the steps
- UI components render the output

Run with:

npm run demo:metrics

## 2. Feedback Form

Demonstrates:

- Simulated user input
- AI summarization
- Workflow orchestration
- Logging + runtime

Run with:

npm run demo:feedback

These apps prove that Magic2U is a functioning design system capable of powering multiple applications with shared components, consistent patterns, and reusable logic.
